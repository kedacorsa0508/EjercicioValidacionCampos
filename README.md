# ValidacionCampos
Este repositorio contiene un ejercicio llamado "EjercicioValidacionCampos", el cual fue propuesto en clase. El objetivo de este proyecto es demostrar cómo llevar a cabo la validación de campos en un programa.

## Pruebas de Validación

Para garantizar el correcto funcionamiento de los métodos implementados en el ejercicio, se han creado pruebas que puedes encontrar en los siguientes paquetes:

- **TestEquipoMaratonProgramacion**: En este paquete, encontrarás las pruebas relacionadas con la validación de campos en el contexto de un equipo de maratón de programación.
- **TestProgramador**: Este paquete contiene las pruebas específicas para la validación de campos relacionados con programadores individuales.
